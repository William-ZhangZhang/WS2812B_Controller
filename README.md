# WS2812B_Controller

## 1-Hardware

硬件部分是我简单设计的一个PCB，
为了减少接线，直接使用电脑USB供电，没有留外部供电接口；
我用的60个灯珠效果很棒，
如果要使用我的这个PCB，灯珠又很多的话，建议自己评估一下电流
